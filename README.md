# WallP – Wallpaper Search Application

WallP is a modern and responsive wallpaper discovery platform that enables users to search, browse, and download high-quality wallpapers using the **Unsplash API**. Built with **HTML, CSS, Bootstrap, and JavaScript**, the application delivers a clean user experience with fast image loading and intuitive navigation.

## ✨ Features

- Search wallpapers by keywords
- Browse wallpapers by categories
- View wallpaper details
- Download high-resolution wallpapers
- Save and manage favorite wallpapers
- Responsive design for desktop, tablet, and mobile devices
- Load More functionality for seamless browsing
- Favorites stored using Local Storage

## 🛠️ Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6+)
- Unsplash REST API
- Local Storage

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/wallp.git
```

2. Navigate to the project folder.

3. Add your Unsplash API Access Key.

```javascript
const ACCESS_KEY = "YOUR_UNSPLASH_ACCESS_KEY";
```

4. Open `index.html` in your browser.
